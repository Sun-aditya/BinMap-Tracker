# BinMap Product Plan

## Product Vision

Build a platform that allows users to instantly locate nearby public dustbins and contribute new locations.

The platform should feel modern, reliable and community-driven.

---

# MVP Goals

A user should be able to:

1. Open BinMap
2. Allow location access
3. View nearby dustbins
4. Select a dustbin
5. Get directions

Additionally:

6. Add a new dustbin
7. Upload image
8. Submit for approval

---

# User Stories

## User Story 1

As a user

I want to locate nearby dustbins

So that I can dispose waste responsibly.

---

## User Story 2

As a contributor

I want to add new dustbins

So that other users can find them.

---

## User Story 3

As a user

I want to report a dustbin as full or damaged

So that information remains accurate.

---

## User Story 4

As an admin

I want to approve dustbin submissions

So that fake locations do not appear.

---

# MVP Features

## Core

- Interactive map
- Geolocation
- Dustbin markers
- Directions
- Dustbin details

## Contribution

- Add dustbin
- Upload image
- Description

## Moderation

- Approve submission
- Reject submission
- Manage reports

---

# Non Goals

Not included in MVP:

- Rewards
- Points
- AI moderation
- Recycling bins
- Public profiles
- Social feed
- Mobile application

---

# Success Metrics

Initial goals:

- 20 mapped dustbins
- 50 users
- 10 contributions
- 100 searches
