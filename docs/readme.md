# BinMap

## Overview

BinMap is a community-driven platform that helps users locate nearby public dustbins and contribute new dustbin locations.

The platform aims to reduce littering by making public waste disposal points easily discoverable.

Initially focused on Chandigarh, BinMap is designed to expand city by city through community contributions.

---

## Problem

People often struggle to find a nearby public dustbin when carrying waste.

This results in:
- Littering
- Carrying waste for long periods
- Poor public cleanliness

There is currently no widely used public platform that maps dustbin locations.

---

## Solution

BinMap provides:

- Interactive map of public dustbins
- Nearby dustbin search
- Directions to dustbins
- Community contributions
- Dustbin status reporting
- Admin verification
